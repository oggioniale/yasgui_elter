# yasgui_elter
A YasGUI SPARQL query editor interface for eLTER-RI

## Clone this repo and run the docker 
```
git clone https://github.com/oggioniale/yasgui.git
cd yasgui_elter
docker-compose up -d
```

Open in your browser: http://localhost:82

to stop

`docker-compose down`
